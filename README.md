# loaderOS | An Operating System Written in 16-bit assembly

LoaderOS Is written in 16-bit assembly, its Lightweight, fast (As for 16bit OS).
LoaderOS Is just an personal Experiment to see how Operating Systems are Running in the background.

 * LoaderOS Supports form 16bit CPUs to 64bit CPUs with Emulation of BIOS Intterupts and 16bit Simulation
 * LoaderOS uses Mimimal RAM becuse 16bit is having limit to 64KB Of Ram
 * LoaderOS Is having App Suport.

All LoaderOS apps are those:

# Notepad
 * LoaderOS Supports Notepad and its having one small file thats is on RAM when you Exit the notepad file is saved to RAM in Shell you can type: list then it shows what was in the file.

# Shell
 * LoaderOS Is having Shell to make OS an Interactive Version, for contect Old Versions of LoaderOS where not having Shell.
 * LoaderOS Shell for now is having those commands: help, list, notepad


# How to run loaderOS
 * NASM (You need to have NASM Installed)
 * QEMU (QEMU Emulator)
 * LD (Linker)
